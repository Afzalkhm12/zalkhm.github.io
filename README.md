## How To Use

To clone and run this application

```bash
# Clone this repository
$ git clone https://github.com/Afzalkhm12/zalkhm.github.io

# Go into the repository
$ cd zalkhm.github.io

# Install dependencies
$ npm install

# Run the app
$ npm start
```
